# lab4
Библиотека с пользовательскими коллекциями

## Структура проекта

```
library_simulation/
├── models/           # Модели данных
│   ├── book.py      # Класс Book
│   └── collections.py # Пользовательские коллекции
├── core/            # Основная логика
│   ├── library.py   # Классы библиотек
│   └── simulations.py # Модуль симуляции
├── utils/           # Вспомогательные функции
│   └── helpers.py   # Утилиты
├── main.py          # Точка входа
└── README.md        # Документация
```

### 1. Пользовательские коллекции
- **BookCollection**: списковая коллекция книг, реализованная через композицию
- **IndexDict**: словарная коллекция для индексации книг по ISBN, автору, году и жанру

### 2. Классы библиотеки
- **Library**: базовая библиотека
- **EnhancedLibrary**: библиотека с выдачей книг (наследник Library)
- **StatisticsLibrary**: библиотека со статистикой операций (наследник Library)

### 3. Симуляция
- **run_simulation()**: функция псевдослучайной симуляции работы библиотеки
- Поддержка seed для воспроизводимости результатов
- 9 типов случайных событий (добавление, удаление, поиск, выдача книг и др.)

## Запуск проекта

1. установлен Python 3.10
2. Скопируйте все файлы в одну директорию
3. Запустите основной файл:
   ```
   python main.py
   ```
4. Следуйте инструкциям в консоли для выбора варианта симуляции
